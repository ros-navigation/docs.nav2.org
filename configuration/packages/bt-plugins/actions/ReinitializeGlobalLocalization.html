ReinitializeGlobalLocalization

Used to trigger global relocalization using AMCL in case of severe delocalization or kidnapped robot problem.

service_name

Type

Default

string

N/A
Description

Service name.

server_timeout

Type

Default

double

10
Description

Server timeout (ms).

Example
