TimeExpired
===========

Node that returns success when a time duration has passed

:seconds:

  ====== ===========
  Type   Default
  ------ -----------
  double 1.0
  ====== ===========

  Description
    The time passed to return success (s).

Example
-------

.. code-block:: xml

    <TimeExpired seconds="1.0"/>
